# Trading Intelligence Dashboard

Portfolio project - dashboard analisis trading rule-based untuk crypto & saham.
Dibangun dengan tech stack gratis.

## Disclaimer
Project ini murni untuk tujuan edukasi dan portofolio. Bukan nasihat keuangan.
Semua sinyal bersifat rule-based, bukan machine learning. Selalu DYOR dan
gunakan manajemen risiko yang ketat sebelum mengambil keputusan trading.

## Fase saat ini: Fase 0-1
- Setup project Next.js + TypeScript + Tailwind
- Live price via Binance WebSocket (gratis, tanpa API key)
- Chart embed TradingView

## Tech stack
- Next.js (App Router) + TypeScript
- Tailwind CSS
- Binance public WebSocket
- TradingView Widget (embed gratis)
- Deploy: Vercel (gratis)

## Cara jalankan lokal
npm install
npm run dev

Buka http://localhost:3000
